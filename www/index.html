An open-source desktop application to manage your private keys

  • Complete control over the sensitive information
  • A vault which never transmits sensitive data
  • Local host storage of the keys to interact with the project

Download

KeyChain is a standalone app for signing transactions and generating key pairs. It stores private keys in an isolated environment where no logger, debugger or spyware can intercept them because of the three-layer security protecting each action of the system. KeyChain supports transactions from and to various blockchains, including Ethereum and Ethereum classic, Litecoin, Bitcoin, and Bitshares.

Key features

  • Independent

    A standalone application to interact with your private keys

  • Secure

    Store your private keys encrypted by AES256 algorithm in the files with privileged access

  • Isolated

    Keep your keys in an isolated environment from your browser, DApp and the local storage

  • Friendly

    The Keychain API easily integrates with any project no matter what language you choose for development
